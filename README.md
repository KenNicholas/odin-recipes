# Odin Recipes

## Project Description
This is a simple recipe website created as part of The Odin Project curriculum. The website will consist of a main index page with links to a few recipes.

## Skills Demonstrated
Upon completing this project, I will have demonstrated:
- **Git & GitHub**: Repository setup, cloning, staging, committing, and pushing
- **HTML**: Document structure, headings, paragraphs, lists, links, and images
- **Command Line**: Navigating directories, creating files, and basic terminal commands
- **Best Practices**: Writing clean code with proper indentation and meaningful commit messages

## What I Learned
This project helps reinforce fundamental web development concepts and version control workflows that are essential for any developer.
